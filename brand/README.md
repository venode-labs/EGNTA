# EGNTA brand

Logo and wordmark for EGNTA, by Venode Labs.

- `egnta-wordmark-light.svg` mark + wordmark, dark ink, for light backgrounds.
- `egnta-wordmark-dark.svg` mark + wordmark, light ink, for dark backgrounds.
- `egnta-monogram.svg` square mark in a rounded tile, for avatars, favicons and OG cards.

All are outlined paths and plain rects (no font dependency, no scripts), so they render
identically anywhere, including the GitHub README via a `<picture>` theme switch.

## The mark

A constructed "E": a bold stem with three arms, the middle arm in the accent teal and
extended past the others. It reads as the letter E and as the product, a read-only
discovery tool that scans the rows of a business and surfaces the one that matters. One
accent gesture, nothing else coloured.

## Type and colour

- Wordmark: Inter Display ExtraBold (the Venode brand face), set tight (-0.02em), outlined.
- Byline: Inter Regular, uppercase, tracked +0.2em.
- Ink: `#0A0E0C` (light bg) / `#F4F6F5` (dark bg).
- Accent: `#00C896`, used once, on the mark's middle arm. Do not add a second accent.

## Usage

Keep clear space around the lockup of at least the height of the mark. Do not recolour
the wordmark, restretch it, detach the mark from the wordmark, or apply effects.

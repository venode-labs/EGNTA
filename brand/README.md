# Egenta brand

Wordmark for Egenta, by Venode Labs.

- `egenta-wordmark-light.svg` dark ink, for light backgrounds.
- `egenta-wordmark-dark.svg` light ink, for dark backgrounds.

Both are outlined paths (no font dependency), so they render identically anywhere,
including the GitHub README via a `<picture>` theme switch.

## Type and colour

- Typeface: Inter Display ExtraBold (the Venode brand face), set tight (-0.025em).
  Outlined to paths, so no font install is needed to render.
- Byline: Inter Regular, uppercase, tracked +0.18em.
- Ink: `#0A0E0C` (light bg) / `#F4F6F5` (dark bg).
- Accent: `#00C896`, used once, on the final letter of the wordmark. One accent
  gesture only; do not add a second.

## Usage

Keep clear space around the wordmark of at least the cap height of the "E". Do not
recolour the wordmark, restretch it, or apply effects. Minimum width ~160px so the
byline stays legible.
